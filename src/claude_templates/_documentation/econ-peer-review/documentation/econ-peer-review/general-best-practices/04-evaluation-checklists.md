# Evaluation Checklists for Peer Review

> Section-by-section checklists for evaluating an economics paper.

## Question and Motivation

- [ ] Research question clearly stated
- [ ] Question important for economics broadly (not just the subfield)
- [ ] Ex ante ambiguity clear (why is the answer not obvious?)
- [ ] Contributions explicitly stated and genuinely novel
- [ ] Contributions numbered (max 3) and each references specific literature

## Literature Review

- [ ] Organized by mechanism/channel, not paper by paper
- [ ] Ends with a gap statement mirroring the contributions
- [ ] Appropriately concise (1-1.5 pages for most journals)
- [ ] Closest antecedents identified and discussed
- [ ] No important missing papers
- [ ] Positive framing of contribution ("we complement" not "they fail to")

## Data

- [ ] All data sources described (provider, coverage, unit, key variables)
- [ ] Sample restrictions justified and documented
- [ ] Treatment/exposure variable precisely defined
- [ ] Summary statistics reported (means, SD, min, median, max) and discussed
- [ ] Missing values and zeros handling explained
- [ ] Potential measurement error in key variables acknowledged
- [ ] Merging procedures and identifiers explained

## Identification Strategy

- [ ] Identifying assumption stated in plain language
- [ ] Assumption credible in this setting — threats discussed
- [ ] Estimator appropriate for data structure (staggered, count, panel)
- [ ] Standard errors clustered at the right level (justified)
- [ ] For DiD: pre-trends shown with coefficients and SEs (not just visual)
- [ ] For DiD (staggered): modern estimator used or TWFE bias acknowledged
- [ ] For IV: first stage reported with F-statistic
- [ ] For IV: exclusion restriction argued and alternative channels addressed
- [ ] For RDD: bandwidth justified, manipulation tests shown
- [ ] Preferred specification without controls reported as benchmark

## Results

- [ ] Main estimates reported with standard errors (not just stars)
- [ ] Economic magnitude discussed (not just statistical significance)
- [ ] Event-study/dynamic effects plots shown
- [ ] Point estimates interpreted (% change, SD interpretation, policy comparison)
- [ ] Numbers in text match tables exactly

## Robustness

- [ ] Alternative estimators tested
- [ ] Sensitivity to sample restrictions shown
- [ ] Placebo tests if applicable
- [ ] Tests genuinely probe identification (not cosmetic)
- [ ] Honestly discussed when alternatives give different results

## Heterogeneity

- [ ] Splits on predetermined (pre-treatment) variables only
- [ ] Each split motivated by economic theory
- [ ] Explicitly labeled as descriptive
- [ ] Null results discussed (not just significant splits)
- [ ] Distinguished from mechanisms section

## Mechanisms

- [ ] Clear theoretical framework for why the effect exists
- [ ] Hypothesis stated before each test
- [ ] Mechanisms operationalized and tested (not just asserted)
- [ ] Distinct from heterogeneity (who vs. why)
- [ ] Competing mechanisms discussed and, where possible, ruled out
- [ ] Null mechanism results reported

## Conclusion

- [ ] Summarizes without repeating the introduction verbatim
- [ ] Policy implications concrete and supported by evidence
- [ ] Limitations acknowledged honestly
- [ ] No new claims not supported by the analysis
- [ ] Appropriately short (1-1.5 pages)

## Writing Quality

- [ ] Standard academic economics register (no hedging, no informal connectives)
- [ ] Paragraphs structured with topic sentences
- [ ] Paper appropriate length for target journal
- [ ] Tables and figures self-contained (readable without text)
- [ ] All tables and figures referenced in text
- [ ] No orphaned exhibits
- [ ] References complete and consistent
