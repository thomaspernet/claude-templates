# Literature Search

Structured search for sources on a research topic.

## Steps

1. **Define query** — Clarify the research question and key search terms. Include synonyms and related terms.
2. **Search** — Search across multiple platforms:
   - Google Scholar for academic papers
   - arXiv for preprints
   - Semantic Scholar for citation graphs
   - Web search for grey literature and reports
3. **Evaluate** — For each source found, assess:
   - Relevance to research question
   - Source tier (1-4, per research-methods rules)
   - Recency and citation count
4. **Summarize** — For selected sources, create a note file in `notes/` with the standard metadata header.
5. **Update bibliography** — Add entries to `references.bib`.
6. **Report** — List sources found, selected, and rejected with brief justification.
