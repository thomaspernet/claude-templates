# Review Draft

Read a document and review it against the style guide.

## Steps

1. **Read the document** — Read the full draft from start to finish.
2. **Check structure** — Does it have clear intro, body, conclusion? Logical flow?
3. **Check against style guide** — Apply all rules from `style-guide.md`:
   - Sentence length (flag any > 25 words)
   - Active voice (flag passive constructions)
   - Jargon (flag undefined technical terms)
   - Heading hierarchy
   - List formatting
4. **Check consistency** — Terminology, formatting, tone throughout.
5. **Report** — Organize findings by section:

```
## Section: {heading}
- [STRUCTURE] {issue and suggestion}
- [CLARITY] {issue and suggestion}
- [STYLE] {issue and suggestion}
```

End with a summary: strengths, top 3 improvements needed, overall readiness (needs work / almost ready / ready to publish).
