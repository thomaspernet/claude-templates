# Review Process Rules

## Self-Review Checklist

Before requesting external review, check:

### Structure
- [ ] Clear introduction with purpose statement
- [ ] Logical section order
- [ ] Each section has a clear point
- [ ] Conclusion summarizes key takeaways

### Clarity
- [ ] No ambiguous pronouns ("it", "this" without clear reference)
- [ ] No undefined acronyms or jargon
- [ ] Complex ideas broken into digestible parts
- [ ] Examples provided for abstract concepts

### Consistency
- [ ] Terminology used consistently throughout
- [ ] Formatting consistent (heading style, list style, emphasis)
- [ ] Tone consistent across sections
- [ ] Date/number formats consistent

### Grammar & Polish
- [ ] No run-on sentences
- [ ] No sentence fragments (unless intentional)
- [ ] Subject-verb agreement
- [ ] Correct punctuation

## Feedback Incorporation

When receiving feedback:
1. Read all feedback before making changes
2. Group feedback by type (structure, clarity, factual, style)
3. Address structural issues first, then details
4. Don't defend — if something confused one reader, it will confuse others
5. Mark addressed feedback as resolved
