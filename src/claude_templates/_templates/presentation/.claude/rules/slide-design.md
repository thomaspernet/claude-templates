# Slide Design Rules

## Slide Anatomy

Every content slide has exactly four layers:

1. **Action title** — A complete sentence stating the slide's conclusion (top of slide)
2. **Subtitle / framing** — Optional one-liner providing context or scope
3. **Body** — The evidence: chart, table, diagram, or concise bullets
4. **Source line** — Data attribution in small font at bottom

## Text Rules

- Action titles: 1 sentence, max 2 lines. Must be a conclusion, not a label.
- Bullets: max 6 per slide, max 6 words per bullet. If you need more, split slides.
- No full paragraphs on slides. If you wrote a paragraph, move it to speaker notes.
- No orphan words (single word on a line by itself) — reflow the text.

## Typography

- Headlines: 28-36pt, bold, sans-serif
- Body text: 18-24pt, regular weight
- Source/footnote: 10-12pt, grey
- Never mix more than 2 font families per deck
- Line spacing: 1.2-1.5x for readability

## Color

- Primary palette: max 3 colors + white/black/grey
- Use color to encode meaning (green = good, red = risk) — be consistent
- Charts inherit the deck palette — no Excel rainbow defaults
- Ensure sufficient contrast (WCAG AA minimum) for accessibility

## Charts and Data Visualization

- Every chart must have a clear title stating the insight, not just the metric
- Label axes and data points directly (no legends if avoidable)
- Flat 2D charts only — never 3D
- Use bar charts for comparison, line charts for trends, pie charts almost never
- Highlight the key data point with color; grey out the rest
- Include units and time periods on every chart

## Layout

- Align all elements to a grid (use PowerPoint guides/snap)
- Consistent margins: minimum 0.5" on all sides
- Left-align text (never center-align body text, only titles if appropriate)
- One visual focal point per slide — don't scatter attention

## What NOT to Do

- No WordArt, clip art, or decorative borders
- No sound effects or auto-advancing transitions
- No walls of text — if it reads like a document, it's not a slide
- No unlabeled charts or unsourced numbers
- No inconsistent formatting between slides
