# Review Deck

Read a presentation outline or slide content and review it against slide design rules.

## Steps

1. **Read the deck** — Read all slide content from start to finish.
2. **Check horizontal flow** — List all titles in sequence. Do they tell a coherent story?
3. **Check slide anatomy** — Each slide has: action title, body evidence, source line.
4. **Check against design rules** — Apply all rules from `slide-design.md`:
   - Action titles (not topic labels)
   - 6×6 rule (bullets and word count)
   - Font sizes (nothing below 18pt)
   - Color consistency
   - Data sourced
5. **Check narrative structure** — Does the deck follow: setup → evidence → conclusion → next steps?
6. **Report** — Organize findings by slide:

```
## Slide {number}: "{title}"
- [TITLE] {issue and fix}
- [CONTENT] {issue and fix}
- [DESIGN] {issue and fix}
- [DATA] {issue and fix}
```

End with: strengths, top 3 improvements needed, overall readiness (needs rework / almost ready / ready to present).
